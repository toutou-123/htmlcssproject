Le PHP et le JavaScript

Avant que je commence à expliquer comment ça fonctionne je vais vous parler de la différence entre le PHP et le JavaScript.

  • Au lieu d'utiliser des tonnes de commandes afin d'afficher du HTML (comme en C ou en Perl), les pages PHP contiennent des fragments HTML dont du code qui fait "quelque chose" (dans ce cas, il va afficher "Bonjour, je suis un script PHP !"). Le code PHP est inclus entre une balise de début qui permettent au serveur web de passer en mode PHP.

  • Le JavaScript est un code qui permet d'afficher des détails à l'utilisateur comme par exemple un bouton qui se met a devenir un arc-en-ciel quand on dirige la souris dessus.

Maintenant que les différences on été présentés, nous pouvons commencer à vous expliquer encore plus de différences sous la forme d'une image.

Comme on peut le constater sur l'image, le javascript peut être directement vu sur la page web alors que le PHP agit de manière invisible au visiteur d'une page web.
