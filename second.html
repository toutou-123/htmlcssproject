Le HTML

Le HTML est la base de toute page internet existante : sans cela la page n'existerait même pas. Pour afficher du texte, du HTML ; pour mettre une image, encore du HTML.
Si le HTML pouvait être confondu avec le corps humain il ressemblerait au squelette.

Pour comprendre comment cela fonctionne voici quelques bases de HTML :

  • Lorqu'une balise est ouverte, elle doit être fermé sinon le code écrit dedans n'est pas pris en compte (exemple : pour fermer < body>, il faut utiliser < /body>)

  • La balise < p> permet d'écrire du texte dans le HTML.

  • < ul> et < li> permettent de créer une liste (comme celle-ci tiens).

  • Une de ces balises est une exception qui n'a pas besoin d'être fermé et c'est < br>. Il coupe un code et peut servir pour espacer sur le HTML.

Si vous voulez savoir comment est codé un site, faites un click droit sur la souris puis choisissez "inspecter".
Vous verrez ainsi tout le code HTML d'un page internet. (j'ai laissé un secret dans cette page trouvez-le avec l'outil inspecter)

Lorsque cliquerez sur inspecter, un onglet s'affichera a droite de la page de recherche, avec vous verrez ainsi le HTML mais aussi le CSS de la page.
