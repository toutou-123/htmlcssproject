Le CSS

Le CSS est la seconde étape pour coder une page web afin qu'elle soit plus belle car sans CSS le fond d'écran de ce site ne serait pas visible.
Une fois encore si on le comparait avec le corps humain ce serait les muscles.

Plus sérieusement, le CSS permet de embellir une page web en faisant un assez grand éventail de choses, dont on à vite fait de se perdre.
C'est pour ça que je vais vous fournir non seulement des repères, mais aussi un site pour apprendre à s'en servir.

  • Pour modifier le HTML, il faut "attraper" ce qu'il faut changer. Prenons la balise "< main>". La ligne de code en dessous permet de modifier la taille et la couleur du texte.

  • Pour selectionner une chose précise dans une balise on fait un espace et on cible ce qu'on souhaite changer. (Note : il faut recréer une nouvelle ligne de code sinon ça ne fonctionne pas)

  • Il existe 2 moyens pour cibler quelque chose dans le html sans mettre son code à l'envers : "class" et "id". Ces deux fonctions sont capable de donner une catégorie à un objet de notre code en HTML l'isolant ainsi pour le modifier sans problème. le "#lien" cible une "id" donné à un objet et".logo-css" cible une "class".

Si vous ne comprenez pas tout ce qui est expliqué, je vous invite à utiliser inspecter pour voir les explications de "class" et "id".

Et pour apprendre le CSS je vous recommande ce lien.
