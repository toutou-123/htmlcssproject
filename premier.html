Pourquoi ce site existe-t-il ?

Ce site existe car la plupart des gens qui veulent se mettrent au codage ne savent pas comment commencer.
Donc vous êtes en chance d'être tombé dessus car vous allez apprendre le codage en douceur.

Le codage est séparé en plusieurs parties : le HTML, le CSS, le PHP et le JavaScript (et non la JavaScript). Chaque partie est important pour la création d'un site.
Mais avec une meilleure maîtrise du codage on peut être capable de créer des jeux vidéos.
Et si vous comprenez pas comment le code fonctionne, internet est votre ami mais il sera probablement anglais.

Cela peut paraitre compliquer lorsqu'on plonge directement dans le codage, c'est pour ça que ce site va être votre ami pour apprendre les bases.
Cliquez sur l'image du html pour commencer ou bien sur une autre image selon votre niveau de connaissance.
